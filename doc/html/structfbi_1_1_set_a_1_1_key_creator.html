Detailed Description

template<typename BoxType, size_t... TIndices>
template<size_t... KeyCreatorIndices>
class fbi::SetA< BoxType, TIndices >::KeyCreator< KeyCreatorIndices >

Handle the creation of keys by using given functors. As we have to handle indices which aren't known at the time of the tree instantiation, we have to add another template class to use different size_t parameter packs.
