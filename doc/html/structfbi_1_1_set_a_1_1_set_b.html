The public interface for the user to start the algorithm.

This function initializes the keys (consisting of intervals in several dimensions) of the intervalSet and the querySet, along with pointers to pass them to separation algorithms. The IndexCalculator has to be used to recalculate the original indices from the pointers.

Parameters:
[in] dataContainer STL Container providing a forward iterator and holding a value_type.
[in] ifunctor A functor to operate on value_type and create a key_type by getting a pair in every indexed dimension.
[in] qdataContainer STL Container providing a forward iterator and holding a qvalue_type.
[in] qfunctors a variadic list of functors to create several key_type objects which will be associated with the original data.
Returns:
An adjacency list implemented by a random-accessable outer container representing the vertices, each holding a container of non-parallel edges.
