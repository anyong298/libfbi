Detailed Description

template<typename BoxType, std::size_t... TIndices>
template<std::size_t Dim, std::size_t Limit>
class fbi::SetA< BoxType, TIndices >::IntersectionTester< Dim, Limit >

Check in if two intervals intersect by comparing in all dimensions till it's either false or all tests evaluated to true.

Check if two objects intersect in all dimensions in [Dim,Limit).

Template Parameters:
Dim The dimension of the _key_ we're primarily working on, this can differ from the actual dimension in value_type.
Limit As we're looking for intersections in the Dim, we have to check for every intersection-pair if they're also matching in the other dimensions, the parameter is used for termination.

If the amount of data is small enough, straightforward checking for two objects can be used. Note that this should only be used after sufficient filtering of possible intersection candidates by the scanners.
