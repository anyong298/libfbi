Detailed Description

Templated helper class, returns the number of functors passed. As we should be able to pass multiple types of functors and even multiple objects per type, we need to calculate how many functors there are. This is needed to calculate the original indices of the boxes from given pointers, as only they are passed around - the re-identification is made by pointer-arithmetic.
