cutoffSize Minimum amount of query and data objects to handle in HybridScanner, if one of them falls below that value, switch to OneWayScanner.
heightCalculator Function pointer to a heuristic which returns the height to use in getApproxMedian.

std::size_t calculate ( bool  isQueryVectorPtr,
const key_type *  objectPtr 
) const

As we pass pointers around during our algorithm, it is necessary to find the corresponding index of the objects when writing pairs as results. To do this, we first save the pointer to the respective first object, using our knowledge of the memory representation a simple pointer subtraction yields the original index.

Parameters:
[in] isQueryVectorPtr We have to keep track if a pointer refers to a box in the first or second input set as that changes its index.
[in] objectPtr a pointer to a key, can be either generated from the first or second type of boxes.
